# MobileJKN-Aspect-Based-Sentiment-Analysis-Dataset
A dataset of 10000 MobileJKN reviews scraped from the Google Play Store using the google_play_scraper library. Manually annotated by 2 researchers on 3 aspect

# Aspects & Counts
- Services: 750 positive & 473 negative reviews.
- Application Performance 470 positive & 1791 negative reviews.
- Registration & Verification 82 positive & 1423 negative reviews.
